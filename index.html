Materia Status

Current status and updates for Materia at UCF.

All Systems Operational

All services are running normally.

Upcoming Maintenance on December 12, 2025:

Materia will be down for extended maintenance starting on Friday, December 12, 2025. We anticipate the system will be back up and running by Tuesday, December 16. During downtime, the application and all associated widgets will be unavailable on both materia.ucf.edu and Webcourses.

What's happening?

The downtime is associated with our newest update, which addresses a big pile of technical debt we've accumulated since the system was launched in 2012. Some of the key changes include a complete rewrite of our server backend from PHP to Python and Django and transitioning from LTI 1.1 to 1.3. LTI is the specification we use to communicate with Learning Management Systems such as Webcourses.

For more information, check our FAQ below. We will update this page with status updates associated with the downtime.
